# Calculadora de Partidas Rankeadas Pro

Projeto completo com menu interativo e sistema de ranking.

## Funcionalidades
- Cadastro de jogadores
- Cálculo automático de saldo
- Classificação por nível
- Ranking ordenado
- Armazenamento em JSON

## Tecnologias
- Node.js
- JavaScript

## Instalação
npm install prompt-sync

## Executar
node app.js
